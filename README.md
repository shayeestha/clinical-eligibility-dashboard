# Clinical Eligibility Dashboard

## Overview

An interactive R Shiny dashboard developed to monitor patient eligibility, enrollment, and recruitment trends in a clinical trial.

The dashboard enables clinical teams and centralized monitoring teams to assess recruitment progress, identify data quality issues, and explore study-level metrics through interactive visualizations.

---

## Features

- KPI Cards
    - Total Subjects
    - Eligible
    - Ineligible
    - Missing Data
    - Inconsistent Records

- Patient Flow Diagram

- Country-wise Distribution

- Site-wise Distribution

- Cumulative Enrollment by Eligibility Status

- Enrollment vs Randomization Trend

- Quarterly Enrollment by Eligibility Status

- Interactive Patient Listing

- Dynamic Filters
    - Country
    - Site
    - Tumor Location
    - Eligibility Status
    - Enrollment Date
